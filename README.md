# resume-coverletter-saas
Production-ready AI Resume &amp; Cover Letter SaaS built with FastAPI, React, PostgreSQL, Stripe, Docker, and modern DevOps practices.

## Completed Features (v0.3.0)

- User registration
- JWT authentication
- Login / Logout
- Refresh tokens
- Email verification
- Password reset
- Google OAuth
- User profile management
- Rate limiting
- Security headers
- Automated backend tests
